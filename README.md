### Embedded Multimodal Focus Monitoring System
A real-time embedded AI system that classifies focus state using pulse sensor data and desk-motion features from an Arduino Nano 33 BLE Sense Rev2. The model was trained using Edge Impulse and deployed on-device with BLE dashboard support.

**Highlights**
- Extracted pulse and IMU-based features from sensor windows
- Trained a lightweight embedded classifier
- Deployed TinyML model directly on Arduino hardware
- Built BLE-based live dashboard for classification output

**Tech:** Arduino, C++, Edge Impulse, TinyML, BLE, IMU Sensors, Pulse Sensor

### Quantum Random Number Generator Uncertainty Analysis
Undergraduate research project studying how uncertainty in beam-splitter-based quantum random number generators affects extraction efficiency, uniform random bit cost, computational cost of Gaussian random number generation, throughput, and latency.

- Analyzed throughput-precision trade-offs in QRNG systems

**Tech:** Python, Probability, Numerical Analysis, Quantum Randomness, Statistical Modeling
